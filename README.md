# TheGame
VideoJuego hecho en Unity version 6000.2.6f2
Es un juego plataforma indie con un toque de atmosfera sombría el cual nosotros controlamos a un hombre que despertó en un bosque totalmente oscuro y sombrío el cual no se sabe que clase de peligros hay.
